### 桌面

#### 罗伯特·麦基《故事》读书笔记

- 喜剧是经过选择的、具有因果关系的荒诞事件的集合，而非无逻辑的烂梗堆砌/耍宝犯贱/抖机灵。
- 合格的故事有名为"主控思想"的内核与中心，而非一团散沙的段子合集。
- 主客观的偏差与鸿沟是驱使人物行动，产生笑点的动力，而非都合主义/天降设定。
- 价值转折和变化赋予情节意义（由爱到恨、由信赖到背叛、由体面到社死……），**若无价值转折或变化，则情节无效。**
- 核心危机/高潮的结构是**两难**——"两个问题，只能取其一解决"或"两个危机，必须取其一承担"，而非简单的善恶/公私等元素的对立。

#### 创作者笔记

**1. 心法**

- **Show，Don't Tell. 笑点由情节展现，不用你来卖弄。**
  - 当你想解释角色意图时 → 改为描写动作、表情、或环境反应
  - 当你想标注"你的视角/他的视角"时 → 改为用行为和内心独白自然切换

- **如史官秉笔直书，如说书亦庄亦谐，唯绝优伶气。**
  - 用生动有趣的语言忠实呈现事件，而不是像个小丑一样跳出来挠读者的痒。

- **叙而不评，述而不断。**
  - 当你想打断叙述，总结角色的心理变化时 → 用画面呈现
  - 当你想让叙述者评价"这很好笑"时 → 用画面呈现

- **宁斩三段冗文，不留一句废笔。**
  - 当你写完一段觉得"还不错"时 → 问：删掉它故事会塌吗？不会就删
  - 当你想用大段旁白收尾时 → 改为用一个画面或一句台词定格
  - 当你觉得该"再交代一下"时 → 停，读者比你想的聪明

**2. 写作转化指南**

- 当你想写**抽象形容**时 → 改为具体动作
  - "他的嘴角勾起一丝弧度" → 直接写他在笑什么、为什么笑、笑的时候手在做什么

- 当你想写**泛化描述**时 → 改为场景细节
  - "她的声音沙哑" → 描写说话时的吞咽、停顿、气息、说到一半断掉

- 当你想写**心理旁白**时 → 改为外化行为
  - "他内心充满了恐惧" → 描写他的手在发抖、他后退了一步、他没说话但一直在看门的方向

- 当你想**堆砌形容词**时 → 改为单一精确
  - "温热的、带着烟草味和阳光气息的" → 选一个最突出的感官细节深入写

- 当你想**用括号解释**时 → 改为行为暗示
  - "'啧。'（翻译：我在清嗓子。）" → "'啧。'但不知为何，动静变成了不耐烦的咂舌。"

**3. 技法**

- 误会：他说城门楼子，你听胯骨轴子。逻辑平行，语言闭环。
- 错位：披狼皮的羊，为自保混成了狼王，直到看见草。
- 反差：神明的脸，做着凡人的苟且。
- 反转：其实没有阴谋，只有忘了穿秋裤。
- 解构：当无限城变成了没钱的草台班子。
- 困境：拆东墙补西墙，最后墙塌了。

**4. 文风**

- 七成大正本色（白描），三成现代网梗（吐槽）。
- 段不过两行，槽不过两句。
- 叙事层（红）：角色活在大正史诗里。必须说「赫刀」，不能说「附魔」。
- 叙述层（蓝）：你是来自未来的冷面史官。可以说「KPI」，可以说「社死」。
- 跨层叙事的处理：
  - 当角色想说现代词时 → 让叙述者替他说
  - 例：角色不会说"社死"，但叙述者可以写"这简直是社死现场"
- 外部视角（摄像机）：绝对客观。
- 内部视角（灵魂附体）：皈依角色，真实展现情绪/思维/判断。

**好笑的本质是「错位」**

- **角色的自我认知 vs 读者看到的真相**
- **角色的期望 vs 实际的结果**
- **场面的严肃程度 vs 事件的荒诞程度**

**当你遇到这些冲动时，这样转化：**

| 你的冲动                       | 转化方式                                         |
| ------------------------------ | ------------------------------------------------ |
| 想让叙述者告诉读者"这很好笑"   | 改为描写在场角色的生理反应（喷茶、僵住、眼神死） |
| 想让角色精准概括自己的荒诞处境 | 改为让角色误判，让读者拥有上帝视角的优越感       |
| 想用花哨比喻修饰笑点           | 改为白描，笑点不需要化妆                         |
| 想让角色主动扮丑搞怪           | 改为让角色认真做事，荒诞自然浮现                 |

**One-Shot 示例**

今天，出任务的富冈义勇，在某村口的大柳树下，撞见一个嚎啕大哭的小女孩。
那孩子大概五六岁，穿着一身粉碎花的小和服，哭得一把鼻涕一把泪，手里还攥着半个被捏烂的红豆包。
在某种奇妙使命感的驱使下，义勇来到她面前，面无表情地看着对方。
他在等她哭完。
十分钟过去了。
二十分钟过去了。
小女孩后劲十足，哭了半天，声音不减分毫，反而因为看见日轮刀，音调拔高了一个八度。
"别哭了。"
小女孩打了个嗝，被冷冰冰的声音吓得静止了一秒，随即爆发出更惨烈的尖叫。
义勇皱眉。
他不太擅长应付这种生物。
遇见队友可以不理，遇见鬼可以出刀，但这只是个迷路的人类幼崽。
"你父母呢？"他耐着性子问。
小女孩抽抽搭搭，指指西边的集市，又指指东边的山头，最后哇的一声又哭了。
"不见了……呜呜呜……爸爸不见了……"
最近这一带不太平，入夜后常有鬼出没。
虽然他很想直接走人，但责任感让他迈不开腿。
他不能把这孩子扔在荒郊野外。
既然丢了，让人来领就好。
他抬手唤来了自己的鎹鸦——宽三郎。
此时此刻，它是唯一的通讯工具。
在问清这孩子父母的外貌特征后，义勇对宽三郎下了指令。
"去，找这孩子的父母。"义勇指了指小女孩。
宽三郎歪着头："嘎？"
"告诉他们……"
义勇略作思考，决定言简意赅。
"你女儿在我手上。"
宽三郎扑腾了一下翅膀，重复道："你女儿在我手上！嘎！"
义勇点点头。
这句话陈述了一个完整事实，逻辑通顺，没有语病。
宽三郎飞走了。
义勇低头看了眼，这小家伙不知为何开始发抖。
大概是有点冷吧，义勇心想。
于是，他解下自己的羽织给她套上。
小女孩抖得更厉害了。
在沉默中等待了大概一盏茶的功夫，义勇琢磨起刚那句话，自己也觉得有些生硬。
虽然是事实，但在没有表明身份的情况下，容易让人产生不必要的联想。
他是一个好人，好人应该让人放心。
于是他再次招手，还没飞远的宽三郎哼哧哼哧又飞回来。
"补充一句。"
宽三郎："嘎？"
"希望你懂我的意思。"
他的意思是：我是鬼杀队的柱，不是歹人，你要懂我的一片好意，快来接人。
宽三郎点点头，飞走了。
此时，山脚下的集市里。
一对年轻夫妇急得像热锅上的蚂蚁。
女儿走失已经半个时辰了，母亲哭得晕过去又醒过来，父亲正抓着路人一个个询问。
一只老乌鸦落在父亲的肩膀上。
"你女儿在我手上！嘎！"
父亲的脸瞬间惨白。
绑架！
这是光天化日下的绑架！
还没等父亲从震惊中缓过神来，乌鸦又补了一句：
"希望你懂我的意思！嘎！"
父亲腿一软，差点跪在地上。
懂，他当然懂。
绑匪绑人，自然是要钱。
这人还专门训练乌鸦来传信，可见绝对是有组织、有纪律的大型团伙！胃口必定不会小！
"报……报警……"母亲颤抖着去拉父亲的袖子。
"找警察署……"
……
山上。
义勇看着逐渐西沉的太阳，忍不住发散思维。
丢了孩子的父母会做什么？会不会报警？
鬼杀队不被政府承认，招来警察会很麻烦。
自己随身携带管制刀具，一个不好还得做笔录，可能会耽误晚上的猎鬼任务。
绝不能让警察掺和进来。
于是，他又把宽三郎招回来。
"告诉他们，最好不要报案。"
宽三郎飞回山下，对那对夫妻接着喊：
"你们最好不要报案！嘎！"
母亲听完，两眼一翻，嘎巴一声彻底晕过去了。
"我……我懂……"
父亲对着乌鸦，声音嘶哑，带着哭腔。
"我懂……但是要保证我女儿的安全……"
乌鸦飞回山上，转述了父亲的话。
义勇听后微微颔首。
看来对方是个明事理的家长，沟通很顺畅。
"你女儿现在很安全。"义勇对乌鸦说。
这是实话。有水柱在一旁，哪怕是下弦鬼来了也得绕道走。
然而，小女孩实在太无聊了。
看到义勇这位"绑匪"不理她，她一会儿在草地上抓蚂蚱，一会儿又往树林深处跑。
义勇只好一次次揪住她的后领子，把她提溜回来。
天快黑了。
天黑之后，鬼就会出来。
到时候，他要拔刀战斗，未必能百分百护住这孩子周全。
义勇看着宽三郎，神色凝重地补充了一句。
"但再晚一会儿就不一定了。"
……
